Plataforma de auxílio acadêmico em TI feito em HTML, CSS e Javascript, criado por Ryan Moraes e Davi Bernado para a Semana de Cultura, Ciência e Tecnologia (SCCT) de 2025 realizada pela ETER de Quintino.

Oferece conteúdos técnicos abrangendo engenharia de software, desenvolvimento web e outras trilhas essenciais da área.
